# 🏛️ BetterSanPedro

A community-led, open-source portal designed to make the government of the **City of San Pedro, Laguna** accessible, transparent, and user-friendly.

This project is a fork of [BetterGov.ph](https://bettergov.ph) and [BetterLB](https://betterlb.org).
